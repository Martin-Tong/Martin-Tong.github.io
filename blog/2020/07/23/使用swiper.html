使用swiper

Jul 23, 2020 • Martin Tong

在页面展示的时候，经常有轮播展示的需求。与自己实现的方式相比，使用成熟、优秀的库可以提高开发体验和实际的展示效果。本篇，介绍的就是这样一个依赖库：swiper.

使用

初始化一个Swiper实例，当页面中有多个实例时可以为每个容器设置不同的clss或id以区分它们。

参数名 类型 必填 描述
swiperContainer HTMLElement or string 必填 Swiper容器的css选择器或者容器的引用
parameters object 可选 Swiper的个性化配置

实例化一个Swiper对象后，该对象会被附加到对用的swiper容器下，可以通过swiperContainer.swiper获取引用。

选项

定制Swiper的主要方式是通过实例化时传入options，包括其提供的一些附加功能（组件）也是以选项的形式实现的。Swiper提供的选项很多，具体可以查看swiper中文api文档。
